About

I made this site originally for a very close friend who never knew what to watch on Plex. It stopped being used after the demise of Plex.

I've expanded it to be more general, and hopefully more useful to more people.

All the movies are stored in a big JSON file movies.json. This file is generated mostly manually, with the help of one of IMDb's APIs to get the IMDb ID.
